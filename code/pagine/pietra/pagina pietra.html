QUESITO 3

Newton si stava divertendo a passeggiare sul letto di un torrente scegliendo le pietre più belle per la sua collezione. Raccogliendone una in particolare, gli venne in mente un interessante indovinello famoso ancora oggi : una pietra pesa un chilo più mezza pietra. Sai dire quanti chili pesa quindi questa fatidica pietra?
